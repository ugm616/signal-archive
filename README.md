# SIGNAL ARCHIVE SYSTEM

A terminal-style idle puzzle game combining Match-3 mechanics, document organization, and signal decoding.

## How to Play

1. Open `index.html` in any modern web browser
2. Match 3 or more tiles in the grid to generate transmission fragments
3. Documents will appear in your inbox - click to read them
4. Signal decoders work while idle, generating documents automatically
5. Organize documents into folders (drag and drop coming soon!)
6. The game saves automatically every 30 seconds

## Features

- **Match-3 Grid**: Swap adjacent tiles to create matches of 3 or more
- **Document System**: Collect and read transmission fragments
- **Idle Decoders**: Multiple frequency decoders that work while you're away
- **Terminal Aesthetic**: Classic green-on-black CRT monitor style
- **Save System**: Auto-save with manual save/load and export/import
- **Progressive Unlocks**: New decoders and mechanics unlock as you play

## Controls

- Click tiles to select and swap them
- Click documents to read their contents
- Use the buttons at the bottom to save/load/reset
- Export/Import your save file for backup

## Hidden Mechanics

This game contains many secrets and hidden systems. Pay attention to:
- Document contents and patterns
- Filing organization effects
- Idle behavior
- Timing of events
- Unlock conditions

## Technical Details

- Pure HTML/CSS/JavaScript (no frameworks required)
- Uses localStorage for saving
- Runs entirely client-side
- Works in Chrome, Firefox, Safari, Edge

## Development

To modify the game:
- `index.html` - Structure and layout
- `style.css` - All visual styling
- `game.js` - Game logic and mechanics

The codebase is structured to support narrative content expansion through the document generation system.
